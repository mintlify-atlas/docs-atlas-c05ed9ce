SmartTube supports casting from mobile devices and offers remote control customization. This guide covers device pairing and remote control configuration.

## Casting from Mobile Devices

Unlike the official YouTube app, SmartTube does not automatically appear on your casting list. You must manually link your mobile device to your TV using a TV code.

<Warning>
SmartTube does not automatically show up when you are in the same WiFi network. Manual pairing is required.
</Warning>

### TV Code Linking Process

Link your smartphone, tablet, or computer to cast videos to SmartTube on your TV.

<Steps>
  <Step title="Open SmartTube Settings">
    On your Android TV:
    1. Launch SmartTube
    2. Navigate to **Settings** from the main menu
    3. Select **Remote control** (2nd option in settings)
    4. A TV code will be displayed on the screen
    
    <Info>
    Keep this screen open while you complete the linking process on your mobile device.
    </Info>
  </Step>
  
  <Step title="Open YouTube on Mobile Device">
    On your smartphone or tablet:
    1. Open the official **YouTube app**
    2. Tap your profile icon (top-right corner)
    3. Go to **Settings**
    4. Select **General** or **Watch on TV**
  </Step>
  
  <Step title="Enter TV Code">
    In the YouTube mobile app:
    1. Tap **Link with TV code**
    2. Enter the code displayed on your TV screen
    3. Tap **Link** or **Connect**
    4. Wait for confirmation
    
    <Tip>
    The TV code is usually 12 digits. Enter it carefully - it's case-sensitive.
    </Tip>
  </Step>
  
  <Step title="Confirm Connection">
    Your devices are now linked!
    - The TV should show a confirmation message
    - Your mobile device is now connected
    - You can now cast videos from your phone to SmartTube
  </Step>
</Steps>

### How to Cast Videos

Once devices are linked, casting is straightforward:

<Steps>
  <Step title="Open SmartTube on TV">
    **Important:** SmartTube must be open on your TV before casting.
    
    <Warning>
    Due to technical limitations, SmartTube cannot automatically wake up the TV. You must open the app first.
    </Warning>
  </Step>
  
  <Step title="Select Video on Mobile">
    On your mobile device:
    1. Open the YouTube app
    2. Browse or search for a video
    3. Start playing the video (optional)
  </Step>
  
  <Step title="Cast to TV">
    1. Tap the **Cast** icon (screen with WiFi waves) in the YouTube app
    2. Select your SmartTube device from the list
    3. Video will start playing on your TV
    4. Use your phone as a remote control
  </Step>
</Steps>

## Remote Control Pairing

SmartTube works with standard Android TV remote controls and supports various input devices.

### Supported Remote Types

<Tabs>
  <Tab title="IR Remote">
    **Infrared Remote Controls:**
    
    - Standard TV remote that came with your Android TV
    - Must point at TV to work (line of sight required)
    - Universal remotes programmed for Android TV
    - No special pairing required - works automatically
    
    <Info>
    Most Android TVs come with an IR remote by default. These work with SmartTube out of the box.
    </Info>
  </Tab>
  
  <Tab title="Bluetooth Remote">
    **Bluetooth Remote Controls:**
    
    - Wireless remotes that don't require line of sight
    - Need to be paired with your Android TV
    - Usually pair through Android TV settings, not SmartTube
    - Include game controllers and wireless keyboards
    
    **How to pair:**
    1. Go to Android TV **Settings** (not SmartTube settings)
    2. Navigate to **Remotes & Accessories**
    3. Select **Add accessory**
    4. Put your remote in pairing mode
    5. Select it when it appears
  </Tab>
  
  <Tab title="Mobile App Remote">
    **Smartphone as Remote:**
    
    - Use your phone as a remote control
    - Download Android TV Remote app
    - Available for Android and iOS
    - Connect via WiFi (same network as TV)
    
    **Setup:**
    1. Install **Android TV Remote Control** app
    2. Open app and select your TV
    3. Enter pairing code shown on TV
    4. Use phone's touchpad to navigate
  </Tab>
</Tabs>

## Customizable Remote Buttons

SmartTube allows customization of certain remote control buttons for quick access to features.

<Accordion title="Button Customization Options">
  You can customize various button behaviors within SmartTube:
  
  **Customizable actions:**
  - Long-press actions on remote buttons
  - Quick access to playback speed
  - Quality selection shortcuts
  - SponsorBlock toggle
  - Subtitle controls
  
  **How to customize:**
  1. Go to **Settings** → **Interface** or **Controls**
  2. Look for **Button customization** options
  3. Assign actions to available buttons
  4. Test your configuration
  
  <Info>
  Available customization options depend on your remote control model and Android TV version.
  </Info>
</Accordion>

<Accordion title="Player Control Buttons">

  |:-------|:--------------|:-------------|
  | **HQ** | Quality selection | Position only |
  | **Speed** | Playback speed | Position only |
  | **Loop** | Autoplay mode | Position only |
  | **Chat** | Live chat | Enable/disable |
  | **Subtitles** | CC controls | Position only |
  
  You can typically hide, show, or reorder these buttons in the settings.
</Accordion>

## Troubleshooting Remote Issues

<Accordion title="Remote not responding">
  **Solutions:**
  
  1. **Replace batteries**: Most common issue with IR remotes
  2. **Check line of sight**: IR remotes need clear path to TV
  3. **Re-pair Bluetooth**: Unpair and pair again in Android TV settings
  4. **Restart TV**: Power cycle your Android TV device
  5. **Check compatibility**: Verify remote is compatible with Android TV
</Accordion>

<Accordion title="Casting doesn't show SmartTube">
  **This is expected behavior!**
  
  SmartTube does not automatically appear on your casting list. You must:
  
  1. **Link devices manually** using the TV code method (see above)
  2. **Open SmartTube first** on your TV before casting
  3. **Keep SmartTube open** - it cannot auto-launch from casting
  
  <Info>
  This is a technical limitation, not a bug. SmartTube cannot automatically wake up or launch like the official YouTube app.
  </Info>
</Accordion>

<Accordion title="TV code not working">
  **Troubleshooting steps:**
  
  1. **Verify code accuracy**: TV codes are case-sensitive, enter carefully
  2. **Check code expiration**: Request a new code if the current one doesn't work
  3. **Update YouTube app**: Make sure your mobile YouTube app is current
  4. **Check WiFi**: Both devices should be on the same network (not required, but helps)
  5. **Restart SmartTube**: Close and reopen SmartTube to generate a new code
</Accordion>

<Accordion title="Phone not controlling TV">
  **After linking, if control doesn't work:**
  
  1. **Verify link**: Check if device is still listed as connected
  2. **Re-link**: Remove the connection and link again with a new TV code
  3. **SmartTube must be open**: Casting only works when SmartTube is running
  4. **Update YouTube**: Update the YouTube app on your mobile device
  5. **Check network**: Ensure both devices have stable internet connections
</Accordion>

## Security and Privacy

<Accordion title="TV Code Security">
  SmartTube uses **one-time connection codes** with limited permissions:
  
  ✅ **Safe features:**
  - Codes have very limited permissions
  - Cannot change your Google account password
  - Only allow video playback control
  - Can be revoked at any time
  
  🔒 **How to revoke access:**
  1. Open [myaccount.google.com/security](https://myaccount.google.com/security)
  2. Find **"Your connections to third-party apps & services"**
  3. Tap **"See all connections"**
  4. Locate **YouTube TV** or **Google Drive**
  5. Select the app → **"Remove access"**
  
  <Info>
  Even though the codes are safe, you can revoke them anytime for full peace of mind.
  </Info>
</Accordion>
