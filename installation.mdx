# Installing SmartTube

SmartTube is easy to install on any Android TV device. Follow the instructions below to get started.

<Warning>
**Important Security Notice**

Do **not** download SmartTube from any app store, APK websites, or blogs. These versions were uploaded by third parties and may contain malware or ads. SmartTube is not officially published on any app store.

Always download from the official sources listed below.
</Warning>

<Info>
**Security Update - January 2024**

The developer's build environment was previously affected by malicious software. All current builds are now clean and scanned with VirusTotal. Public keys may have been compromised, so new keys are now in use.

**No action required** - SmartTube uses one-time connection codes with limited permissions. However, if you want extra peace of mind, you can revoke access:

1. Open [myaccount.google.com/security](https://myaccount.google.com/security)
2. Find "Your connections to third-party apps & services"
3. Tap "See all connections" and locate YouTube TV or Google Drive
4. Select the app → "Remove access"

All current downloads are safe and verified.
</Info>

## Device-Specific Instructions

### Chromecast with Google TV

Chromecast with Google TV blocks app installation by default. Follow these additional steps:

<Steps>
  <Step title="Enable Developer Options">
    On your Chromecast:
    1. Open the side menu and go to **Settings** > **System** > **About**
    2. Scroll down to **Android TV OS build**
    3. Click it repeatedly until you see a message saying you're steps away from being a developer
    4. Continue clicking until developer mode is enabled
    
    ![Enable Developer Options](https://www.androidpolice.com/2021/02/07/how-to-sideload-any-apk-on-the-chromecast-with-android-tv/#install-the-apk)
  </Step>
  
  <Step title="Enable Unknown Sources">
    1. Go back to the main **Settings** page
    2. Select **Apps** > **Security & Restrictions** > **Unknown sources**
    3. Enable the toggle for **Downloader by AFTVnews** (or whichever app you're using to install)
  </Step>
  
  <Step title="Follow general installation steps">
    Now you can proceed with any of the [installation methods](#installation-methods) above.
  </Step>
</Steps>

### Xiaomi Devices with Chinese Firmware

<Warning>
Xiaomi's Chinese firmware may block installation of the beta version. The international firmware is not affected.
</Warning>

**Solutions:**

<AccordionGroup>
  <Accordion title="Solution 1: Use Stable Version (Recommended)">
    Install SmartTube's **stable version** instead of beta. The stable version is not blocked by Chinese firmware.
    
    Use URL: `kutt.it/stn_stable`
  </Accordion>
  
  <Accordion title="Solution 2: Flash International Firmware">
    Flash the international firmware for your device. Search for a guide specific to your Xiaomi device model.
  </Accordion>
  
  <Accordion title="Solution 3: Factory Reset Method (2020 or older devices)">
    For devices from 2020 or before:
    1. Perform a factory reset
    2. Install SmartTube beta **before** doing any system updates
    3. After SmartTube is installed, you can safely update your system
    
    SmartTube should continue working after system updates.
  </Accordion>
</AccordionGroup>

## Troubleshooting Installation Issues

<AccordionGroup>
  <Accordion title="Installation fails immediately">
    **Possible causes:**
    
    1. **Disk space is full** - Clear up storage space on your device
    2. **APK file didn't download correctly** - Try downloading again
    3. **Unknown sources not enabled** - Enable installation from unknown sources for your file manager/downloader app
    
    **Solution:** Clear storage space and try downloading/installing again.
  </Accordion>
  
  <Accordion title="App crashes when opening">
    **Cause:** SmartTube is installed to external storage (SD card).
    
    **Solution:** Uninstall and reinstall SmartTube to **internal memory**, not to an SD card or external storage.
  </Accordion>
  
  <Accordion title="Cannot find the installed app">
    **Solution:** Check your app launcher. SmartTube should appear in your list of installed apps. If not, try reinstalling.
  </Accordion>
  
  <Accordion title="'Parse error' during installation">
    **Cause:** Corrupted APK download.
    
    **Solution:** Delete the APK file and download it again from the official sources.
  </Accordion>
</AccordionGroup>

## Updating SmartTube

<Note>
SmartTube has a **built-in updater**. You only need to follow the installation procedure once.
</Note>

After launching SmartTube, it will automatically check for updates and notify you with a changelog. To update:

1. Wait for the update notification (appears a few seconds after launch)
2. Follow the prompts to download and install the update
3. Or manually check: **Settings** > **About** > **Check for updates**

**If update installation fails:**
- Your disk space might be full - clear up storage space
- Try updating again from **Settings** > **About** > **Check for updates**
